Introduction

To use Vayu Hub, simply paste the following loadstring into your script execution box:

local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/IRONJAM1/Vayu-Hub-UI/refs/heads/main/source.lua"))()

ü™ü Creating a Window

Initialize the library by creating a main window.

local Window = Library:CreateWindow({ Title = "My Script Name", ThemeColor = Color3.fromRGB(0, 145, 255), Logo = "90392555143990", -- Optional UseBlur = true })

üß© Adding Elements

Learn how to add tabs, buttons, toggles, and more.

Create Tab

local MainTab = Window:CreateTab("Main", "6034509993")

Create Toggle

MainTab:CreateToggle("Auto Farm", false, function(Value) print("Auto Farm:", Value) end)

Create Slider

MainTab:CreateSlider("WalkSpeed", 16, 500, 16, function(Value) game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value end)
